# Novo Tratamento para Retinopatia Diabética

Estudos recentes indicam que a terapia com siRNA reduziu a expressão de genes inflamatórios na retina do paciente com DM2. O Ibuprofen trata episódios agudos de dor de cabeça relatados por 15% dos indivíduos durante os ensaios. A veiculação via nanoemulsão lipídica mostrou-se promissora para transpor a barreira hemato-retiniana, limitando os efeitos adversos sistêmicos.
